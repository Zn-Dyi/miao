So What is This About?

There is a continuing need to show the powerof CSS. The Zen Garden aims to excite, inspire, and encourage participation. To begin, view some of the existinng designs in the list. Requirements

Where possible, we would like to see mostly CSS1 &: 2 usage. CSS 3 &: 4 should be limited to widely-supported elements only, or strong fallbacks should be provided. the CSS ZenGarden is about functional, practical CSS and not the latest bleeding-edge tricks viewable by 2% of the browsing public. The only real requirement we have is that your CSS validates.
